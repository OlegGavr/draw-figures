Draw Figures

Write the command(s)

Example: line -p [20, 20] [50, 70] -c rgb(240, 100, 50)

All commands

  • line -p [50, 50] [100, 100] -c rgb(255, 0, 0) -w 2;
  • rectangle -p [150, 50] [200, 100] -c rgb(0, 255, 0) -b rgba(255, 0, 100, 0.3) -w 2;
  • triangle -p [400, 50] [450, 100] [350, 100] -c rgb(0, 0, 255) -b rgba(0, 255, 0, 0.3) -w 3;
  • circle -p [75, 250] -r 25 -c rgba(0, 0, 255, 0.8) -b rgba(255, 255, 0, 0.5) -w 4;
  • ellipse -p [500, 230] -r1 50 -r2 25 -c rgb(255, 0, 100) -b rgba(255, 100, 100, 0.3) -w 5;
  • star -p [300, 150] -n 5 -r1 50 -r2 25 -c rgb(0, 255, 255) -b rgba(0, 0, 255, 0.3) -w 2;

Tip 1: if there are several commands, then put a separator `;` at the end of each of them.

Tip 2: the X-coordinate belongs to the range from 0 to {{widthCanvas}}, Y-coordinate - from 0 to 200.
